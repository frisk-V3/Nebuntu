### Project Overview
**Nebuntu** は Ubuntu ベースで開発中のカスタムOSプロジェクトです。  
「学タブ（学校配布タブレット）でも動く、軽量で美しいデスクトップ体験」を目指しています。

> [!TIP]
> Nebuntu は **教育端末でも扱いやすい OS** をコンセプトに設計されています。
## Concept
**「学タブに、自由と創造性を。」**
